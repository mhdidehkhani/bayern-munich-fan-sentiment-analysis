# Bayern Munich Fan Sentiment Analysis using Social Media Data

This project analyzes the sentiment of FC Bayern Munich fans on social media platforms such as Reddit, YouTube, and Twitter (X).

## Project Overview
The aim of this project is to explore how fan emotions expressed online relate to the team’s performance in football matches.

## Data Sources
- Reddit
- YouTube
- Twitter (X)

The dataset includes fan comments related to Bayern Munich matches between 2022 and 2025.

## Methodology
1. Data collection from social media platforms
2. Text preprocessing and cleaning
3. Sentiment analysis using the VADER model
4. Statistical analysis (Correlation and ANOVA)

## Tools
- Python
- Pandas
- NLTK / VADER
- Matplotlib
- SciPy
